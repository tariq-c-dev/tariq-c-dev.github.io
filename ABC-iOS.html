ABC iOS

ABC iOS is no longer maintained, but source code is available!

ABCs is educational app to help children connect the written Alphabet with the spoken Alphabet, using the power of an interactive soundboard. Click or touch a letter and hear it!

This app can no longer be downloaded from TestFlight, but you can download the code here and open in Swift Playgrounds on an iPad or Mac. The code is very simple; feel free to read or use it!
