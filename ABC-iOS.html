ABC iOS

ABC iOS is no longer maintained, but source code is available!

ABC iOS is an educational app to help children connect the English written Alphabet with the spoken Alphabet, using the power of an interactive soundboard. Click or touch a letter and hear it!

This app can no longer be downloaded from TestFlight, but you can download source code here and open in Swift Playgrounds or Xcode.
